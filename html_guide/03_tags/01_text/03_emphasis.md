# 3. 강조 태그

## 3-1. `<strong>`

```html
<!-- 의미적 강조 (강한 중요성) -->
<strong>중요한 정보</strong>
```

## 3-2. `<em>`

```html
<!-- 의미적 강조 (부드러운 강조) -->
<em>강조하고 싶은 부분</em>
```

## 3-3. `<b>`

```html
<!-- 단순 시각적 강조 (의미 없음) -->
<b>굵게</b>
```

## 3-4. `<i>`

```html
<!-- 단순 스타일링 / 외국어 등 강조 없음 -->
<i>기울임</i>
```
