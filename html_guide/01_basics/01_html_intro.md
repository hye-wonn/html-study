# 1. HTML 기본 개념

## 1-1. HTML이란?

-   웹 페이지의 구조를 정의하는 마크업 언어
-   **태그(tag)**를 이용해 구성 → 웹 브라우저는 이를 해석하여 화면에 표시

## 1-2. 주요 특징

-   파일 확장자: `.html`
-   **문서 유형 선언 (Doctype)**: `<!DOCTYPE>`

## 1-3. HTML 실행 방법

### 1-3-1. 웹 브라우저에서 실행

-   저장한 `.html` 파일을 **웹 브라우저 (Chrome, Edge, etc.)** 에서 열어 자동 실행 가능

### 1-3-2. VSCode에서 실행

-   Live Server 확장 프로그램 설치 → 저장 즉시 웹 브라우저에서 확인 가능
-   실시간 미리보기 및 변경 사항 자동 반영
